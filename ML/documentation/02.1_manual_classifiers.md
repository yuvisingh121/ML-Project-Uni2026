## 3. Richiamo teorico: Classificatore Naive Bayes

Il classificatore Naive Bayes si basa sul teorema di Bayes:

P(Y | X) = [ P(X | Y) * P(Y) ] / P(X)

Nel contesto della classificazione, il termine P(X) è identico per tutte le classi e può essere ignorato.  
Il classificatore assegna quindi un'osservazione alla classe che massimizza:

P(Y = y | X) ∝ P(Y = y) * Π P(X_i | Y = y)

dove:
- Y è la variabile target (nel nostro caso: 0 = non diabetico, 1 = diabetico)
- X è il vettore delle feature osservate
- X_i è la i-esima feature

L’ipotesi “Naive” consiste nel considerare le feature condizionalmente indipendenti dato Y.

Poiché le feature del dataset sono discrete o binarie, utilizziamo la versione discreta del Naive Bayes.  
Per evitare probabilità nulle si applica il Laplace smoothing:

P(X_i = x | Y = y) = ( N(X_i = x, Y = y) + 1 ) / ( N(Y = y) + k )

dove:
- N(X_i = x, Y = y) è il numero di osservazioni con quella combinazione
- N(Y = y) è il numero totale di osservazioni della classe y
- k è il numero di valori possibili della feature X_i

Il classificatore finale calcola due punteggi:

Score_0(Y=0) = P(Y=0) * Π P(X_i | Y=0)  
Score_1(Y=1) = P(Y=1) * Π P(X_i | Y=1)

La classe predetta è quella con punteggio maggiore.

## 7. Costruzione del classificatore Naive Bayes manuale

Il classificatore calcola:

Score_1(Y=1) = P(Y=1) * Π P(X_i | Y=1)

La classe assegnata è quella con punteggio maggiore.
